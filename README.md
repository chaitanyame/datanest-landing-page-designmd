# DataNest

> Cloud-Native Document Database

**DataNest** is a cloud-native document database designed for modern applications. It combines the flexibility of JSON documents with the power of SQL-like querying, real-time sync, and automatic scaling — all with a developer-first experience.

## 🗄️ Features

- **Document Model** — Store JSON documents with dynamic schemas and nested data
- **SQL-Like Queries** — Rich query language with joins, aggregations, and full-text search
- **Real-Time Sync** — Subscribe to query changes with WebSocket-based live updates
- **Auto-Scaling** — Horizontal scaling with automatic sharding and replication
- **ACID Transactions** — Multi-document transactions with snapshot isolation
- **Full-Text Search** — Built-in Lucene-based search engine with relevance scoring
- **SDKs** — Native clients for JavaScript, Python, Go, Rust, Java, and .NET
- **Edge-Ready** — Multi-region replication for low-latency global access

## 🛠 Tech Stack

- **Core Database:** Rust (storage engine), C++ (query optimizer)
- **Consensus:** Raft-based replication
- **Search:** Tantivy (full-text), PostgreSQL (catalog)
- **API:** gRPC, REST, GraphQL
- **Infra:** Kubernetes, Helm, Terraform providers
